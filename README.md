# code_agent
Автоматизированная агентная система, реализующая SDLC внутри GitHub. Система имитирует работу разработчика и ревьюера: анализирует задачи, вносит изменения в код, создает Pull Request, запускает CI/CD, анализирует результаты проверок и принимает решение о завершении работы либо повторе цикла разработки.
